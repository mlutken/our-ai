
C++: Generic uri class
======================

C++: Please write a versatile C++23 class called "uri" for representing and manipulating both file system paths and URLs in a unified way. It shall extend on the functionality of std::filesystem::path to support schemes (e.g., http, https, file, ftp), query parameters, fragments, and authentication, while maintaining compatibility with local file paths. So I must support the same operations as std::filesystem::path and it should have an instance of std::filesystem::path as a private member to help in the implementation. Also we should follow Cpp Core Guidelines and know best practices. All class and function names should be lowercase using underscores (snakecase) when multiple words in an identifier.
